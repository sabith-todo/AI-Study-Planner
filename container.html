IOT Project

IoT-based smart farming monitoring (software only) is a digital system that uses simulated sensor data and software tools to monitor farm conditions like soil moisture, temperature, humidity, and weather. The system displays data on a web or mobile dashboard, analyzes it using predefined rules, and generates alerts or irrigation decisions automatically. It helps farmers monitor crops, save water, and improve productivity without using physical IoT hardware, making it ideal for academic and demo projects. 🌱

Academic Project

This academic application includes an admin allocation page for managing users and resources, an announcement page for sharing important updates, and a home page that provides easy navigation to all modules. It features a secure login page, material and notes pages for accessing study resources, a timetable page for scheduling, and role-based services to ensure appropriate access for admins and students. The system is designed to support effective academic management and enhance the overall learning experience.

Todo Rover Project

I am developing a task-based autonomous rover that is capable of performing assigned tasks automatically using programmed logic and intelligent decision-making. The rover operates with minimal human intervention and is designed to navigate, monitor, and execute tasks efficiently. This project focuses on autonomy, reliability, and practical application in real-world environments.
